# Trading Journal

A full-stack trading journal application built with:

- React
- TypeScript
- TailwindCSS
- Node.js
- Express
- MongoDB

## Features

• User authentication  
• Dashboard statistics  
• Trade tracking  
• Win rate calculation  
• P/L tracking  

## Tech Stack

Frontend:
- React
- Vite
- TypeScript
- TailwindCSS

Backend:
- Node.js
- Express
- MongoDB
